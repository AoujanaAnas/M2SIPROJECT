Optimisation Problme

Information du projet

Dans ce projet en a travailller sur deux problemes d'optimisation.

Probleme 1 :Dans ce probleme, on considère n tâches à exécuter sur une seule ressource afin de minimiser l’objectif 1 : 1|| ΣUi.

Probleme 2 :Dans ce probleme, on considère n tâches à exécuter sur une seule ressource afin de minimiser l’objectif 2 : 1|| ΣαiEi+ βiTi.

Polynomial Algorithm.

A polynomial-time algorithm is an algorithm whose execution time is either given by a polynomial on the size of the input.

Dynamic Algorithm.

Dynamic programming is both a mathematical optimization method and a computer programming method. The method was developed by Richard Bellman in the 1950s and has found applications in numerous fields, from aerospace engineering to economics.

Meta-Heuristic Algorithm

A metaheuristic is a high-level problem-independent algorithmic framework that provides a set of guidelines or strategies to develop heuristic optimization algorithms.

Informations à propos application

Probleme 1

Les tâches sont toutes disponibles à l’instant 0 avec chacune une durée opératoire pi et une date de livraison di. Noter qu’après le démarrage de la première tâche aucun temps mort n’est autorisé sur la ressource (i.e. chaque solution réalisable doit commencer à t=0 et se terminer à t=Σpi). Noter ici qu’il possible d’utiliser l’algorithme de Moore-Hodgson (AMH) pour résoudre ce problème. En conséquence, et à titre purement illustratif, il faut inclure la comparaison de l’algorithme AMH avec DP et GVNS pour ce problème.

Probleme 2:

Les tâches sont toutes disponibles à l’instant 0 avec chacune une durée opératoire pi et une date de livraison di. Noter qu’après le démarrage de la première tâche aucun temps mort n’est autorisé sur la ressource.

Plus d'information Concernant la méthode

Polynomial: utiliser Moore-Hodgson algorithm.

Dynamic: utiliser le fonction f(J) = min { f(J\{k}) + U(k) } .

Meta-heuristic: Utiliser 3 structures de voisinage (2-opt , swap , insertion) pour chaque phase de GVNS.
